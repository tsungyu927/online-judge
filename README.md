# Online Judge

## Platform

- [LeetCode](https://leetcode.com/)

## Language

- Kotlin
- Golang

## How to run

### Kotlin

#### 1. Compile to a JAR

```shell
kotlinc <PATH_TO_THE_FILE>.kt -include-runtime -d <PATH_TO_OUTPUT_FILE>.jar
```

#### 2. Run the JAR

```shell
java -jar <PATH_TO_OUTPUT_FILE>.jar
```
